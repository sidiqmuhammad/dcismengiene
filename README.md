# dcismengiene
Communication bridge (driver) between Proxmox host operating system and Dell PowerEdge server hardware

Tested in PVE 9.1.1 (Debian 13 Trixie) + Dell PowerEdge R760

```dpkg -i dcism_5.3.0.0-3289.ubuntu22_amd64.deb```

```dpkg -i dcism-osc_7.3.0.0_amd64.deb```

```systemctl enable --now dcismeng```
